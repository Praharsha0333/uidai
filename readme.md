# UIDI – Aadhaar Resilience System

This project analyzes Aadhaar enrolment and update stress patterns
to study system resilience using data-driven indicators.

## Files
- final_notebook_uidai.ipynb – Main analysis notebook
- app.py – Web dashboard application
- aadhaar_dashboard_data.csv – Processed dataset

## How to Run
pip install -r requirements.txt
python app.py
